Session 2 · Job Aid

Physical Count

Monthly stock verification · Count before transactions

⚠

Critical: count BEFORE daily transactions

On counting day, complete the physical count before recording any new receipts, issues, or adjustments. This captures an accurate snapshot of what is physically in the cold room.

Steps

1

Open Physical Count

Tap Physical Count → select programme → enter counting date → tap Proceed.

2

Work through every product

Do not skip any product — count them all, even if the quantity is zero. Select each product, then each batch.

3

Enter the counted quantity

  • Screen shows expected quantity (what the system thinks you have)
  • Count the doses physically and enter the actual quantity
  • If counts match → no discrepancy flagged
  • If counts differ → system flags the discrepancy
  • Tap Save for each batch
4

Continue or pause

  • Continue → count the next batch or product
  • Done for now → save progress and resume later
  • Submit the full count when all products are covered
ℹ

Discrepancies do not auto-adjust SOH

The system records the discrepancy but does NOT change the stock on hand automatically. If the discrepancy is real, record a separate Adjustment (e.g. reason = Missing or Found in Store) to correct the stock.

⚡

Count with a partner

One person counts and calls out the quantity; the other enters it into the app. Reduces errors and provides a second check.

✓ Practice checklist

  • Open Physical Count → select programme and today's date
  • Find a product and enter the SAME quantity as system SOH — confirm no discrepancy
  • Count a second product and enter a quantity that DIFFERS by 5 — observe the flag
  • Tap Continue to move to the next batch
