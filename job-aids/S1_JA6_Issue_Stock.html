Session 1 · Job Aid

Issue Stock

Record outgoing vaccines · Always apply FEFO

✓

FEFO rule — always issue earliest expiry first

FEFO = First Expiry, First Out. Select the batch at the TOP of the batch list (earliest expiry date). Skip red (expired) batches and take the next one down.

Steps

1

Open Issue

Tap Issue → select programme → enter date of issue → tap Proceed.

2

Find product & select FEFO batch

  • Tap the vaccine in the commodity list
  • Select the top batch in the batch list (earliest expiry = FEFO)
  • Skip any batch shown in red (expired)
3

Enter issue details

FieldWhat to enterRequired?
Issue ToSelect the destination facility from the dropdownYES
Reason'Issuing' is pre-selected — leave as isYES
QuantityNumber of doses — cannot exceed available SOHYES
4

Submit & confirm

  • Tap Submit → confirm on the prompt
  • SOH reduces automatically
  • Continue for next product or Done for now to exit
⚠

Quantity cannot exceed SOH

If you enter more than the available stock on hand, the system shows an error and blocks submission. Check Facility Stock first if unsure.

✓ Practice checklist

  • Open Issue → select programme and date
  • Find the product from the Receive practice
  • Select the FEFO batch (top of list, earliest expiry)
  • Select any destination facility, enter quantity 20, submit
  • Try entering a quantity greater than SOH — observe the error
  • Verify SOH decreased by 20 in Facility Stock
